## Notas / Observações

- **Regressão vs run-001**: adicionar os 3 bots com threshold 0.60 piorou o return de +29.8% para +18.8% e reduziu trades de 677 para 467.
- Diagnóstico: Bonnie v1 com threshold 0.60 rejeita ~60% dos sinais de Clyde, incluindo muitos válidos. A v1 não está suficientemente calibrada para o regime do período.
- Conclusão: Bonnie v1 em modo "gate estrito" (0.60 global) é prejudicial — é necessário re-treinar com features mais ricas e thresholds por regime.
- Cash drag mantém-se ~30%: o cap de 20% por posição limita o número de posições simultâneas.
- Esta run motivou as melhorias arquitecturais da run-003: retrain Bonnie (v2), thresholds por regime, cap 10%, adds, value trailing.
