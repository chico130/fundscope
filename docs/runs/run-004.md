### M3 — Learner 7 anos com fitness adaptativa ao regime

Fitness reescrita para ser sensível ao regime dominante:

- **bull_trending** (>60% dias bull): `alpha × max(0.5, sharpe)` — prioriza alpha vs SPY
- **weighted** (>20% dias bear): `bull_pct × bull_fitness + bear_pct × bear_fitness`
- **lateral/fraco**: `sharpe × profit_factor`

onde `bear_fitness = calmar × max(0.4, 1 - |DD| / 15)`.

O período 2019–2026 tem 1734 dias bull / 1858 total = **93.3% bull** → regime `bull_trending` → fitness = alpha × sharpe_floor.

## Notas / Observações

- **Trade-off esperado:** v3 foi treinada para maximizar alpha ao longo de 7 anos heterogéneos. Naturalmente fica suboptimal para o window 2024-2026 que foi quase exclusivamente bull. v2 foi treinada nesse mesmo window — overfitou nele.
- **Assimetria R:R:** Stop 1.75×ATR + TP 4.25×ATR implica R:R ~2.4:1. WR cai de 52% → 38% mas as vitórias são maiores. Com PF=1.34 e WR=38%, a win média é ~3.5× a loss média. Este perfil é mais robusto em regimes de tendência.
- **Alpha 7yr -1.1pp:** Pela primeira vez o sistema iguala praticamente o SPY numa janela de 7 anos. O baseline (v2 params) ficava em -86pp de alpha. Salto de 84.9pp.
- **Convergência rápida (2 ciclos):** O Learner convergiu em 2 ciclos porque os valores óptimos estavam "nos extremos" do espaço — o ciclo 1 encontrou-os todos e o ciclo 2 confirmou que não existiam melhorias incrementais.
- **atr_tp_mult é o param mais impactante:** No Ciclo 1, atr_tp_mult foi o 2.º a ser melhorado (cur=3.0 → best=4.25, f=-29.6 vs -97.5 baseline). A combinação com atr_stop_mult 1.75 cria o perfil assimétrico.
- **Regime dominante:** 93.3% dos dias foram bull_trending no período 2019-2026. A fitness adaptativa usou bull_fitness em quase todos os ciclos. Para testar os ramos bear/lateral seria necessário treinar num período com mais volatilidade regime.
- **Próximos passos sugeridos:** (1) Validar v3 params num stress-test 2019-2026 com breakdown por ano para confirmar que 2022 (bear) melhora; (2) considerar Bonnie retrain com labels compatíveis com o novo R:R (target ATR×4.25); (3) explorar se `atr_stop_mult_momentum` deve convergir para 1.75 também.
